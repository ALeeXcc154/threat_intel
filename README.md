# 威胁情报生成器

> 输入 URL，一键生成标准化 Word 威胁情报月报。

## 简介

在PM日常信息收集及交付中，手动整理多篇威胁情报文章、统一格式排版、生成汇报文档是一项重复且耗时的任务。

**自动化月报生成器** 将这个过程自动化——只需粘贴情报文章的 URL 链接，工具会自动完成网页内容抓取、结构化解析、多篇文章的标题合并，最终输出一份排版精美的 Word 文档，可直接用于月度汇报或内部归档。

## 核心能力

- 🔗 **批量解析** — 支持一次性粘贴多个 URL，逐个抓取处理
- 🔀 **智能合并** — 自动识别相同大标题的文章，将内容归并到一起
- 📄 **标准排版** — 输出格式统一、排版规范的 Word 文档（标题、正文、缩进、行距、参考链接）
- 📑 **自动生成目录** — 内置 TOC 域代码，Word 中可一键更新

## 快速开始

### 前提条件

确保系统已安装 Python 3.8+。

### 安装

```bash
# 下载至本地
cd threat_intel

# 安装依赖
pip install -r requirements.txt
```

### 运行

```bash
python main.py
```

启动后访问 [http://localhost:8000](http://localhost:8000) 即可使用。

## 使用教程

1. **粘贴 URL** — 在左侧文本框中粘贴威胁情报文章的链接（每行一个，支持多个）
2. **生成情报** — 点击「生成情报文档」按钮，系统开始解析
3. **查看进度** — 右侧面板实时显示处理日志和各 URL 的解析结果
4. **下载文档** — 解析完成后，点击「下载文档」即可获取 `.docx` 文件

## 输出文档效果

生成的 Word 文档按以下结构组织：

| 元素     | 样式                             |
| -------- | -------------------------------- |
| 目录     | 自动生成，支持 Word 内刷新       |
| 大标题   | 红色加粗，同一标题跨文章自动合并 |
| 小标题   | 加粗，每个小标题独立分页         |
| 正文     | 首行缩进 2 字符，1.5 倍行距      |
| 参考链接 | 斜体灰色字体                     |

## 技术栈

- **后端**: [FastAPI](https://fastapi.tiangolo.com/) + [Uvicorn](https://www.uvicorn.org/)
- **文档生成**: [python-docx](https://python-docx.readthedocs.io/)
- **前端**: HTML + [TailwindCSS](https://tailwindcss.com/) + [Axios](https://axios-http.com/)

## 项目结构

```
.
├── main.py            # FastAPI 应用入口，处理路由和文件响应
├── threat_intel.py    # 核心处理器：URL抓取 → 内容解析 → 标题合并 → Word生成
├── index.html         # 前端界面（含完整 JS 业务逻辑）
└── requirements.txt   # Python 依赖列表
```

## 已知限制

- 当前仅支持解析特定威胁情报平台的文章格式（基于 `__INITIAL_STATE__` 数据结构）
- 不支持 PDF 或纯文本文章的解析
